# Framework

A lightweight, modular and beginner-friendly framework for Roblox.

Unlike many frameworks, Framework does not try to hide Roblox behind layers of abstraction. Instead, it provides a clean foundation that helps you organize your game while keeping everything understandable.

## Philosophy

- ✅ Everything is optional.
- ✅ No forced architecture.
- ✅ No hidden magic.
- ✅ Easy to learn.
- ✅ Easy to scale.
- ✅ Roblox first.

Framework adapts to your project—not the other way around.

---

## Features

- 📦 Services
- 🎮 Controllers
- 🌐 Net
- 📡 Signal
- ⚙️ ECS
- 🧩 Modular architecture
- 📖 Built-in documentation

---

## Installation

Copy the installer into Roblox Studio's Command Bar and run it.

The framework will automatically generate the complete folder structure.

---

## Getting Started

Open:

Framework/_START_HERE

(or `_INDICE` if you kept that name.)

From there you'll find where everything belongs.

---

## Philosophy

Framework is intentionally simple.

It does **not** try to become another Knit.

It focuses on helping developers finish games instead of learning a complex framework.
